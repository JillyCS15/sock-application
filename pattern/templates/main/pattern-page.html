    Completeness for Class Properties
    Sejauh mana entitas pada KG memiliki property-property yang diwajibkan untuk suatu class. Misalnya pada class manusia, entitasnya mesti punya jenis kelamin, golongan darah, dsb. Secara umum, untuk suatu class, jika property A, B, C, & D ada semua, maka entitas tersebut dianggap lengkap.
    Code SHACL Pattern Description
    1
    		Cek keberadaan suatu property pada entitas sebagai instance dari suatu class secara umum (general).
    2
    		Pattern untuk mengecek kardinalitas minimum dari suatu property pada suatu instance class yang ada di Wikidata KG. Class didefinisikan pada %% CLASS-URI %% . Property-property yang diharuskan ada didefinisikan pada %% PROPERTY-01 %%, %% PROPERTY-02 %%, hingga %% PROPERTY-NN %% sesuai kebutuhan.
    3
    		Cek kardinalitas minimum dari suatu property pada entitas sebagai instance dari suatu class. Nilai kardinalitas dari property diasosiasikan sebagai batas minimum untuk suatu entitas dikatakan komplet/lengkap.
